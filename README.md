# AnimeHay-Enhanced-Helpers

A personal UI enhancement script suite designed to streamline navigation and improve watchability on AnimeHay.

## Included Scripts

- **Animehay to Anilist Sync** (`Animehay2Anilist.user.js`): Syncs your watch progress from AnimeHay directly to your Anilist tracking.
- **AnimeHay Episode Tracker** (`animeHay_epTracker.user.js`): Keeps track of episodes watched and provides a better episode navigation interface.
- **AnimeHay Smart Skipper** (`animehaySmartSkipper.user.js`): Automatically skips intros, outros, and unneeded parts of the video stream.

Please refer to the individual READMEs for each script to find more detailed installation, setup, and usage instructions:
- `README_Animehay2Anilist.md`
- `README_animeHay_epTracker.md`
- `README_animehaySmartSkipper.md`
