# 💖 AnimeHay Episode Tracker

<img src="../assets/tracker.png" width="50%" alt="Episode Tracker">

Never lose your place again! This script perfectly highlights your next episode in gold. ✨

### ✨ Magic Features
- **Smart Highlighting:** Instantly spots your current AniList progress and highlights the next episode button. 🎯
- **Auto-Scrolls:** Gently scrolls the right episode into view so you don't have to hunt for it!
- **Cute Mini-GUI:** A tiny debug window that fades away when you scroll over it. 👻
