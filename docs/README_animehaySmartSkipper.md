# ⏩ AnimeHay Smart Video Skipper

![Smart Skipper](assets/skipper.png)

Tired of manually skipping 1:30 minute anime openings? Let this script do the skipping for you! 🚀

### ✨ Magic Features
- **Global Skip DB:** Save opening/ending timestamps for any anime you watch. 📚
- **Auto-Skip:** Seamlessly zips you past the boring parts without lifting a finger!
- **Floating UI:** A cute, draggable mini-menu to easily add new skip rules on the fly. 🎛️
